FAQ

How many glasses does one bottle represent?

The default model uses about five 150 ml pours per 750 ml bottle.

Should I add a safety margin?

Yes. Many events add a small buffer to avoid running out near the end of service.

Can I model only one wine style?

Yes. Set one style share to 100% and the others to 0%.
